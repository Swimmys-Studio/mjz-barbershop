# MJ'Z Barbershop Website Prototype

A simple static website prototype for MJ'Z Barbershop.

## Files

- `index.html` - main page structure
- `styles.css` - design and layout
- `script.js` - mobile menu interaction
- `logo.jpg` - provided shop logo

## How to use on GitHub Pages

1. Create a new GitHub repository.
2. Upload all files from this folder.
3. Commit and push.
4. In GitHub, go to **Settings > Pages**.
5. Under **Build and deployment**, choose **Deploy from a branch**.
6. Select the `main` branch and `/root` folder.
7. Save.
8. GitHub will publish the site and give you a public URL.
