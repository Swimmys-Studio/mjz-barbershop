About The Shop

A barbershop built on family, style, and community

MJ'Z Barbershop is a public-facing barbershop website prototype built to showcase the shop's identity, make it easy for customers to connect on social media, check reviews, and book appointments.

The design uses the shop logo, bold typography, and a clean dark-and-gold color palette to match the barbershop brand.
